#کلاس ها

2. کلاس Major نشان دهنده یک رشته دانشگاهی میباشد
3. کلاس Student نشان دهنده یک دانشجو در دانشگاه است
4. کلاس Professor نشان دهنده یک استاد در دانشگاه است
5. کلاس Course نشان دهنده یک درس در دانشگاه است مانند درس « ریاضی عمومی 2 »
